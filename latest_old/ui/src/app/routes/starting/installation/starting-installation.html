Let's Install It

Before You Start

Make sure you have the following installed globally.

yo
npm install -g yo

Scaffi

Time to build up a scaffold project Open a CMD prompt Browse to your projects folder Make a folder for your new project

md 'Project Folder'

Go into your newly created folder.

cd 'Project Folder'

Run the yeoman scaffold command

yo scaffi

Follow the prompts to scaffold up your new project. When you're done, it will do an npm install/jspm install so it while take a bit before you can launch the app.
